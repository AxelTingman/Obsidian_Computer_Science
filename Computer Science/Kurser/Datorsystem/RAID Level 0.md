# RAID Level 0
***
  
• RAID Level 0, also known as drive spanning,  
provides improved performance, but no redundancy.  
– Data is written in blocks across the entire array  
– The disadvantage of RAID 0 is in its low reliability (låg  
tillförlitlighet).
